## 📝 技術筆記

### 短影音平台發布策略

**平台選擇優先順序：**
1. **Instagram Reels** - 目標受眾18-34歲，觀看率13.08%
2. **TikTok** - 高互動率2.34%，演算法強
3. **YouTube Shorts** - 搜尋優勢，SEO友好

**技術規格：**
- 格式：MP4
- 解析度：1080x1920（9:16比例）
- 時長：30-45秒（最佳效果）
- 檔案大小：最大4GB

**內容優化策略：**
- 前3秒抓住注意力
- 清晰大字體字幕
- 匹配的背景音樂
- 明確的CTA設計

### SendGrid聯絡人管理

**分類系統：**
- 已註冊：成功加入平台的餐廳
- 有興趣：回應邀約但未註冊
- 無回應：未回覆邀約的聯絡人
- 取消訂閱：主動取消的聯絡人

**標籤管理：**
- 餐廳類型標籤
- 地區標籤
- 回應狀態標籤
- 註冊時間標籤

### 內容行銷整合策略

**文章內容應用：**
- 提取關鍵論點和數據
- 整合到短影音腳本
- 更新餐廳邀約論述
- 建立內容資料庫

**多平台發布：**
- 統一品牌訊息
- 適配平台特性
- 建立發布時程
- 追蹤效果數據
