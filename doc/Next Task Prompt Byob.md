### 🚨 優先級 1：Yelp 費城 BYOB 餐廳資料爬取

**目標：**
建立 Yelp 爬蟲腳本，收集費城 BYOB 餐廳候選名單（80-150 家）

**預估時間：** 2-3 小時

---

#### **任務 1.1：Yelp 爬蟲腳本開發（90 分鐘）**

* [ ] **技術環境設定**
  * 確認 Python 環境和相關套件（BeautifulSoup, Selenium, requests）
  * 設定 User-Agent 和請求延遲（避免被封鎖）
  * 準備測試環境

* [ ] **爬蟲腳本開發**
  * 開發 Yelp 搜尋結果爬蟲
  * 實作多關鍵字搜尋功能：
    * "BYOB Philadelphia"
    * "bring your own wine Philadelphia"
    * "bring your own bottle Philadelphia"
    * "corkage fee Philadelphia"
    * "BYO Philadelphia"
  * 抓取餐廳基本資訊：
    * 餐廳名稱
    * 地址
    * 電話（如果可見）
    * 網站連結
    * 料理類型
    * Yelp 頁面連結

* [ ] **資料去重和清理**
  * 實作餐廳名稱和地址比對去重
  * 建立初始信心度評估邏輯
  * 輸出為 CSV 或 JSON 格式

**技術注意事項：**
```
✅ 只抓取公開基本資訊
✅ 設定合理的請求間隔（3-5 秒）
✅ 實作錯誤處理和重試機制
❌ 不抓取評論、評分、照片
❌ 不使用 Yelp API（需要付費）
```

---

#### **任務 1.2：資料初步分析（30 分鐘）**

* [ ] **資料品質檢查**
  * 檢查抓取到的餐廳數量
  * 分析資料完整性（有多少筆有電話、網站等）
  * 識別需要手動驗證的問題資料

* [ ] **初步分類**
  * 按料理類型分類（義式、法式、亞洲、美式等）
  * 按區域分類（Center City、Rittenhouse 等）
  * 標記高可信度候選餐廳
