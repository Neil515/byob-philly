## 📝 技術筆記

### SendGrid Email發送系統的技術要點

**SendGrid Domain Authentication設定：**
1. **DNS記錄設定**：在Namecheap設定CNAME記錄（em5468, s1._domainkey, s2._domainkey）
2. **DNS傳播等待**：24-48小時讓DNS記錄完全傳播
3. **Domain驗證**：在SendGrid後台點擊「Verify」按鈕
4. **測試發送**：使用byobmap.com網域發送測試Email

**Email發送程式設計：**
1. **Python SendGrid SDK**：使用sendgrid套件建立發送程式
2. **個人化內容**：針對不同餐廳類型客製化Email內容
3. **發送間隔**：每封Email間隔5-10分鐘避免被標記為垃圾郵件
4. **錯誤處理**：完整的錯誤處理和重試機制

**邀約Email模板設計：**
1. **主旨確定**：「[合作邀約] 你是歡迎客人自帶酒水的餐廳嗎？邀請你免費加入台北BYOB餐廳地圖」

3. **價值主張**：清楚說明加入BYOBMAP的優勢

**追蹤系統建立：**
