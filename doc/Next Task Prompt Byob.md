## 📝 技術筆記更新

### 短影音 CTA 策略

* 前 3 秒抓注意力：「台北哪些餐廳可以自帶酒？」
* 結尾 CTA：「留言或填表單爆料 → 抽餐酒券」
* 平台發布：Instagram Reels、TikTok、YouTube Shorts

### 顧客爆料名單表單架構

1. 餐廳名稱（必填）
2. 地址或地點（必填）
3. BYOB 條件（免費 / 開瓶費 / 不清楚）
4. 餐廳特色或推薦理由（選填）
5. 姓名/暱稱（選填）
6. 聯絡方式（Email/IG，用於抽獎通知）

### 誘因設計

* 每月抽獎（餐酒券 / 酒杯 / 禮券）
* 「BYOB 探店王」排行榜，公布活躍貢獻者

### 社群互動腳本

* IG Story 投票：「你知道哪些餐廳可以自帶酒嗎？」
* IG Reels Hook：「台北居然有這麼多餐廳可以自帶酒？！🍷」
* IG 貼文 CTA：「留言或填表單爆料餐廳 → 抽餐酒券」
