# 🍷 BYOB 專案文檔

## 📋 專案概述

BYOB (Bring Your Own Bottle) 是一個自帶酒水餐廳推薦平台，目前運營兩個獨立專案：

### 🇹🇼 台北 BYOB（主要專案）
- **定位**：台北自帶酒水餐廳推薦平台
- **階段**：核心系統完成，推廣與酒商合作階段
- **策略**：抽獎激勵 + 多平台推廣

### 🇺🇸 費城 BYOB（新專案）
- **定位**：Yelp 的 BYOB 專業補充平台
- **階段**：資料收集完成，進入社群驗證階段
- **策略**：多平台資料收集 + Reddit 社群驗證 + 榮譽系統驅動

## 🎯 核心功能說明

### 台北專案核心功能

**重複檢查系統**
* 自動觸發檢查機制
* 智能相似度計算（名稱 + 地址）
* 地址相同強制判定重複
* 審核管理介面一鍵操作

**抽獎系統**
* Mersenne Twister 隨機演算法確保公平性
* 自動記錄推薦者參與
* 中獎/未中獎 Email 通知
* 獎項：一獎（酒商電子禮券）、二獎（進口紅白酒杯）

**推薦通知系統**
* 審核通過自動觸發
* HTML Email 模板
* 防重複發送機制
* 包含抽獎資訊

## 💡 關鍵策略差異

### 台北模式
```
抽獎激勵 → 一次性參與 → 物質獎勵 → 成本高
```

### 費城模式（升級）
```
榮譽系統 → 持續參與 → 專業認同 → 可持續
```

**為什麼改變？**
1. **成本考量**：海外專案無法負擔持續物質獎勵
2. **文化差異**：美國用戶更重視專業認同
3. **可擴展性**：榮譽系統可無限擴展
4. **長期價值**：建立真正的專家社群

**互補價值：**
- 台北：驗證核心技術和產品概念
- 費城：驗證商業模式和國際擴展性
- 互相學習：費城經驗可回饋台北
