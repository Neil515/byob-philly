# 🍷 BYOB 專案開發文檔

## 📋 專案概述

BYOB (Bring Your Own Bottle) 是一個餐廳資訊平台，讓消費者可以找到支援自帶酒水的餐廳。本專案使用 WordPress + WooCommerce 作為後端，並整合自定義的餐廳業者會員系統。

### 🎯 專案目標
- 建立完整的餐廳業者會員系統
- 提供餐廳資料管理功能
- 實現自動化餐廳資料收集
- 建立專業的餐廳邀約追蹤機制
- 建立 SEO 優化與內容行銷系統
- 整合SendGrid Email發送系統
- 建立餐廳業者註冊通知系統
- 建立專業內容行銷策略
- 建立短影音行銷系統

## 🏗️ 技術架構

- **後端**: WordPress 6.4.2 + WooCommerce 8.5.2
- **主題**: Flatsome Child Theme

- **Email發送**: SendGrid + Domain Authentication + Python SDK

- **手動觸發功能**：提供 `byob_trigger_member_notification()` 函數供直接加入流程使用

- **專業Email格式**：HTML格式設計，包含後台管理連結和注意事項

**Google Places API限制的完全突破**
- **網格搜尋策略**：將搜尋區域劃分為多個小網格，每個網格點獨立搜尋
- **多中心點搜尋**：使用12個台北主要區域作為搜尋中心點，完全覆蓋台北市
- **搜尋記錄機制**：使用JSON檔案記錄已搜尋的網格點，避免重複搜尋
- **斷點續傳功能**：支援中斷後從上次位置繼續搜尋，提升系統穩定性
- **智能去重複**：使用place_id避免同一餐廳被重複收集
- **進度追蹤**：詳細的搜尋進度顯示和統計，每10個網格點自動保存進度
