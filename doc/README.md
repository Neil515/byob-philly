# 🍷 BYOB 專案文檔

## 📋 專案概述

BYOB (Bring Your Own Bottle) 是一個自帶酒水餐廳推薦平台，目前運營兩個獨立專案：

### 🇹🇼 台北 BYOB（主要專案）
- **定位**：台北自帶酒水餐廳推薦平台
- **階段**：核心系統完成，推廣與酒商合作階段
- **策略**：抽獎激勵 + 多平台推廣

### 🇺🇸 費城 BYOB（新專案）
- **定位**：Yelp 的 BYOB 專業補充平台
- **階段**：資料收集完成，進入社群互動階段
- **策略**：多平台資料收集 + Reddit 社群驗證 + 榮譽系統驅動

## 💡 關鍵策略差異

### 台北模式
```
抽獎激勵 → 一次性參與 → 物質獎勵 → 成本高
```

### 費城模式（升級）
```
榮譽系統 → 持續參與 → 專業認同 → 可持續
```

**為什麼改變？**
1. **成本考量**：海外專案無法負擔持續物質獎勵
2. **文化差異**：美國用戶更重視專業認同
3. **可擴展性**：榮譽系統可無限擴展
4. **長期價值**：建立真正的專家社群

**互補價值：**
- 台北：驗證核心技術和產品概念
- 費城：驗證商業模式和國際擴展性
- 互相學習：費城經驗可回饋台北
