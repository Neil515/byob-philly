# 🎁 BYOB 抽獎活動規劃

## 📅 活動概覽

### 活動名稱
**「BYOB 推薦達人抽獎活動」**

### 活動目標
- 提升顧客推薦參與度
- 建立忠實用戶社群
- 增加平台曝光度
- 收集更多優質餐廳資料

### 活動期間
- **每月循環**：每月1日-月底
- **抽獎時間**：每月最後一天 20:00
- **公布時間**：抽獎後24小時內

---

## 🎯 參與資格

### 基本資格
1. **成功推薦餐廳**：透過顧客推薦表單提交餐廳資訊
2. **餐廳通過審核**：推薦的餐廳成功上架 BYOB 平台
3. **提供聯絡方式**：留下有效的 Email 或社群媒體帳號

### 特殊資格
- **連續推薦獎勵**：連續3個月都有推薦的用戶，額外獲得1次抽獎機會
- **優質推薦獎勵**：推薦的餐廳獲得高評分的用戶，額外獲得1次抽獎機會
- **社群分享獎勵**：在社群媒體分享推薦成功通知的用戶，額外獲得1次抽獎機會

### 排除條件
- 重複推薦同一餐廳（以餐廳名稱和地址判斷）
- 提供虛假或無效的餐廳資訊
- 惡意刷推薦行為

#### 一等獎（1名）

- **價值**：情感價值 + 社群認同

### 特殊獎品（季度活動）
- **季度大獎**：高級酒器套組（NT$ 2,000-3,000）
- **年度大獎**：米其林餐廳體驗券（NT$ 5,000-8,000）

## 📋 活動流程

### 每月循環流程

#### 第1-28日：推薦收集期
1. **顧客提交推薦**：透過 Google 表單推薦餐廳
2. **自動回覆確認**：立即發送感謝信和抽獎說明
3. **餐廳審核上架**：推薦成功通知包含抽獎提醒
4. **參與資格確認**：系統自動記錄符合資格的推薦者

#### 第29-30日：抽獎準備期
1. **參與者名單整理**：確認最終參與名單
2. **抽獎工具準備**：使用隨機數生成器
3. **獎品準備**：確認獎品庫存和合作餐廳

#### 第30日 20:00：抽獎執行
1. **直播抽獎**：在社群媒體進行直播抽獎
2. **結果記錄**：詳細記錄抽獎過程和結果
3. **中獎者通知**：24小時內發送中獎通知

#### 第1-3日：結果公布期
1. **社群公布**：在 Instagram 和 Facebook 公布結果
2. **Email 通知**：發送中獎通知給得獎者
3. **獎品發放**：安排獎品寄送或領取

## 🛠️ 技術實作規劃

### 參與者名單管理系統

#### 資料結構設計
```php
// 參與者資料表
lottery_participants {
    id: 唯一識別碼
    recommender_name: 推薦者姓名
    recommender_email: 推薦者 Email
    restaurant_name: 推薦餐廳名稱
    restaurant_id: 餐廳文章 ID
    submission_date: 提交日期
    approval_date: 審核通過日期
    month: 參與月份
    extra_chances: 額外抽獎機會
    status: 參與狀態
}
```

#### 核心功能
1. **自動記錄**：餐廳上架時自動記錄推薦者
2. **資格檢查**：檢查重複推薦和無效資料
3. **額外機會**：計算連續推薦和優質推薦獎勵
4. **名單匯出**：每月抽獎前匯出參與者名單

### 抽獎工具設計

#### 抽獎演算法
```javascript
// 抽獎邏輯
function drawLottery(participants) {
    // 1. 計算總抽獎機會
    let totalChances = 0;
    participants.forEach(participant => {
        totalChances += 1 + participant.extra_chances;
    });
    
    // 2. 生成隨機數
    let randomNumber = Math.floor(Math.random() * totalChances) + 1;
    
    // 3. 找出中獎者
    let currentChance = 0;
    for (let participant of participants) {
        currentChance += 1 + participant.extra_chances;
        if (randomNumber <= currentChance) {
            return participant;
        }
    }
}
```

#### 抽獎介面
- **直播抽獎**：使用 OBS 直播抽獎過程
- **結果展示**：即時顯示中獎者資訊
- **錄影存檔**：保存抽獎過程影片

## 📊 活動監控與分析

### 關鍵指標 (KPI)

#### 參與度指標
- **月參與人數**：每月參與抽獎的推薦者數量
- **推薦轉換率**：提交推薦到通過審核的比例
- **重複參與率**：連續參與抽獎的用戶比例

#### 社群指標
- **直播觀看數**：抽獎直播的觀看人數
- **社群互動**：抽獎相關貼文的按讚、分享、留言數
- **品牌曝光**：抽獎活動帶來的品牌曝光度

#### 業務指標
- **餐廳收集量**：透過抽獎活動收集的餐廳數量
- **用戶留存率**：參與抽獎後繼續使用平台的用戶比例
- **推薦品質**：抽獎活動收集的餐廳品質評分

### 數據收集方法

#### 自動化收集
- **Google Analytics**：追蹤網站流量和用戶行為
- **社群媒體分析**：監控貼文互動和曝光
- **Email 分析**：追蹤郵件開啟率和點擊率

#### 手動收集
- **用戶回饋**：透過問卷收集用戶意見
- **餐廳回饋**：收集合作餐廳的意見
- **團隊觀察**：記錄活動執行過程中的觀察
